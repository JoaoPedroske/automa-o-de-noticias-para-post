# automa-o-de-noticias-para-post
Automação de Conteúdo para Instagram com IA — Desenvolvi um sistema no n8n que busca notícias diárias via NewsAPI, usa GPT-4 para transformar em posts otimizados com hashtags e call to action, e entrega por email automaticamente. Aplicação prática de automação, IA e marketing digital.
